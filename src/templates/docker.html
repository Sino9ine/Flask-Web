สรุปความรู้เกี่ยวกับ Docker

Docker คืออะไร?

Docker คือเครื่องมือสำหรับ **Containerization** ที่ช่วยให้เราสามารถห่อหุ้มแอปพลิเคชันและสภาพแวดล้อมที่จำเป็นทั้งหมดไว้ใน Container ทำให้แอปพลิเคชันทำงานได้เหมือนกันในทุกที่

การสร้างและตรวจสอบ Docker

  • การสร้าง Image: เขียน `Dockerfile` แล้วใช้คำสั่ง `docker build -t sino9inebtw/image:tag .`
  • การสั่งทำงาน: ใช้คำสั่ง `docker run -d -p 80:80 sino9inebtw/image:tag`
  • การตรวจสอบ: ใช้คำสั่ง `docker ps` เพื่อดู Container ที่กำลังรันอยู่
