สรุปความรู้เกี่ยวกับการใช้ Git และ GitHub

Git คืออะไร?

Git คือ **ระบบควบคุมเวอร์ชัน (Version Control System)** ที่ช่วยให้ติดตามและจัดการการเปลี่ยนแปลงของโค้ดได้ ซึ่งสำคัญมากในการทำงานร่วมกันเป็นทีม

ขั้นตอนการใช้งาน GitHub เบื้องต้น

  1. `git init`: เริ่มต้น Repository ในเครื่องเรา
  2. `git remote add origin [URL]`: เชื่อมต่อกับ Repository บน GitHub
  3. `git add .` และ `git commit -m "ข้อความ"`: บันทึกการเปลี่ยนแปลง (Commit)
  4. `git push -u origin main`: อัปโหลดโค้ดขึ้น GitHub
